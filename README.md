## Useful commands

### Node.js

- update NodeJS

```batch
npm install --global --production npm-windows-upgrade
npm-windows-upgrade

```

- Create new sample React APP

```batch
npx create-react-app first-react-site
cd first-react-site
npm start
```

- Prepare to procudtion

```batch
npm run build
```
